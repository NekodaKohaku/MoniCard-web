## 🇹🇼 MoniCard Web (魔力卡網頁版)

👉 **[開啟 MoniCard Web 線上工具](https://nekodakohaku.github.io/MoniCard-web/)**

MoniCard Web 是一款網頁版的 MoniCard（魔力卡）裝置管理工具，透過瀏覽器的 Web Bluetooth 功能即可直接連接。它支援查看裝置概要、名片設定、媒體傳輸（圖片與影片上傳）、標籤設定、設備設定、管理已接收卡片、本地韌體更新以及應用程式設定。

### 🎯 專案亮點
* **無需微信 (WeChat)：** 專門為沒有微信帳號、或不想使用官方小程式的使用者設計。打開網頁就能直接連接並操作您的魔力卡。

### 💻 瀏覽器與硬體支援
本工具依賴 **Web Bluetooth API**，請確保您的使用環境符合以下條件：
* 🚨 **重要前提：** 您的電腦必須**具備內建藍牙功能**，或插上**藍牙接收器**（並確認藍牙功能已開啟）。
* 🟢 **支援：** 電腦桌機版（Windows / macOS / Linux / ChromeOS）的 **Google Chrome** 或 **Microsoft Edge**。
* ❌ **不支援：** Safari、Firefox，以及**所有 iOS 裝置（iPhone / iPad）**（因 Apple 系統限制，即使在 iPhone 下載 Chrome 也無法使用）。

### ⚠️ 免責聲明 (Disclaimer)

本專案為獨立開發的第三方開源工具，**使用者須自行承擔使用風險**。
由於韌體更新與媒體傳輸等功能會直接將資料寫入硬體，對於因使用本軟體而導致的裝置損壞（變磚）、資料遺失或任何硬體異常，開發者不承擔 any 法律與技術責任。

## 🇺🇸 MoniCard Web

👉 **[Launch MoniCard Web App](https://nekodakohaku.github.io/MoniCard-web/)**

MoniCard Web is a browser-based tool designed to connect and manage your MoniCard (also known as "MoliCard" / 魔力卡) devices via Web Bluetooth. 

It provides full support for:
* **Devices Overview**
* **Card** (Business card editing)
* **Media** (Image & video uploads)
* **Tags** configuration
* **Device Settings**
* **Received Cards** synchronization
* **Firmware** (Local firmware updates)
* **App Settings**

### 🎯 Key Highlights
* **No WeChat Required:** Built specifically for users who do not have a WeChat account or prefer not to use the official applets. Open the page and manage your MoniCard directly.

### 💻 Browser & Hardware Requirements
This tool requires the **Web Bluetooth API**. Please ensure your environment meets the following:
* 🚨 **Prerequisite:** Your computer must **have built-in Bluetooth functionality** or use a **Bluetooth dongle** (and make sure Bluetooth is turned on).
* 🟢 **Supported:** **Desktop** versions of Chromium-based browsers such as **Google Chrome** and **Microsoft Edge** (Windows / macOS / Linux / ChromeOS).
* ❌ **Not Supported:** Safari, Firefox, and **all iOS devices (iPhone / iPad)** (Due to Apple's restrictions, Chrome on iOS will not work either).

### ⚠️ Disclaimer

This is an independent, third-party open-source utility. **Use this tool entirely at your own risk.** 
Since features like firmware updates and media transfers write data directly to the device hardware, the developers are not responsible for any device bricking, data loss, or hardware damage caused by using this software.

## 🇯🇵 MoniCard Web

👉 **[MoniCard Web ツールを開く](https://nekodakohaku.github.io/MoniCard-web/)**

MoniCard Webは、ブラウザのWeb Bluetooth 機能を使ってMoniCard（中国語名：魔力卡）デバイスに接続し、管理できるウェブツールです。

以下の機能に対応しています：
* **デバイス概要** (Overview)
* **カード** (名刺設定)
* **メディア** (動画・画像アップロード)
* **タグ** 設定
* **デバイス設定**
* **受信カード** 管理
* **ファームウェア** (ローカルアップデート)
* **アプリ設定**

### 🎯 プロジェクトの特徴
* **WeChat（微信）不要：** WeChatアカウントを持っていない方や、公式ミニプログラムを使いたくない方向けに特別に設計されています。ブラウザを開くだけで、すぐにMoniCardを操作できます。

### 💻 対応ブラウザと動作環境（ハードウェア要件）
本ツールは **Web Bluetooth API** を使用しているため、環境に制限があります：
* 🚨 **必須条件：** お使いのPCに**Bluetooth機能が内蔵されているか、Bluetoothレシーバー（ドングル）が接続されており**、Bluetoothが有効になっている必要があります。
* 🟢 **対応：** **PC（デスクトップ）版** の **Google Chrome** または **Microsoft Edge**（Windows / macOS / Linux / ChromeOS）。
* ❌ **非対応：** Safari、Firefox、および**すべてのiOSデバイス（iPhone / iPad）**（Appleの制限により、iPhone版のChromeでも動作しません）。
